## Mod Introduction

<span style="color: #A7E8FE;">Starry Night Revelry</span> is a <span style="color: #66CBFE;">Friday Night Funkin'</span> mod developed based on <span style="color: #A7E8FE;">Codename Engine</span>.
It is currently in an unfinished state, stay tuned for our future updates!

## Story Background

BF and GF went up the mountain to watch the sunset, but they got separated due to some accident. While BF was searching for GF, he discovered a snow house...

## Installation Instructions

1. Make sure you have Codename Engine installed
2. Copy the `Starry-Night-Revelry` folder to the `mods` directory of Codename Engine
3. Launch Codename Engine and select "Starry Night Revelry" from the mod list
4. Click "Play" to start the game!

## System Requirements

- Latest version of Codename Engine (1.0.1)
